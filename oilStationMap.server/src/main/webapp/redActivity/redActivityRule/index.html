• 活动时间为2018年10月01日0:00-2019年10月01日0:00；

  • 活动三种玩法：

    • （1）玩法一：添加油站拿红包

    • 活动期间，点击【我的】--->【添加油站】，填写您想要添加的加油站，并输入相关信息，比如名称，地址，油价等即可获取红包；每个用户每天仅可领取一次【添加油站】红包。

    （2）玩法二：纠正油价拿红包

    • 活动期间，点击【今日油价】或者【油价地图】，对当前不准确或者未及时更新的油价进行纠正即可获取红包；每个用户每天仅可领取一次【纠正油价】红包。

    （3）玩法三：邀请好友送红包

    • 活动期间，将【油价地图】分享给微信好友，并邀请好友完成1，2步骤即可获取红包

  • 本次活动获得的红包，仅在本次活动期间有效，活动结束后即失效；

  • 关于活动的问题，可以在【油价地图】小程序中，【我的】页面中，联系【我的客服】；

  • 在法律法规许可的范围内，【油价地图】拥有本活动的最终解释权；

  • 该活动与微信官方无关，与手机生产商如苹果公司无关；
